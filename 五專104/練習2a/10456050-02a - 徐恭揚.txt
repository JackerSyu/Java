例如:
a5322121a  (正確)
a          (不正確)
Ans:
class Main {

        String s="aa";
        int i = s.length();
        
        if(i>=2){
          System.out.println("正確");
        }else{
          System.out.println("不正確");
        }
    }
}

class Main {
    public static void main(String[] args) {
        String s="a13545b";
        int i = s.length();
        
        String fst=s.substring(0,1);  
        String lst=s.substring(i-1,i);
        
        if(i>=2 && fst.equals(lst)){
          System.out.println("正確");
        }else{
          System.out.println("不正確");
        }
    }
}

(5)
字串長度至少為5, 內容只能有數字及小數點.
字串內容可以沒有小數點, 但如果內容有小數點, 只能有1個, 而且小數點之後一定要有其他數字.
例如:
54221     (正確)
543.1     (正確)
5321.     (不正確)
5.3.1     (不正確)
Ans:
class Main {
    public static void main(String[] args) {
        String s="11111.";
        int j=s.length();
        int cnt =0;
        boolean flag=true;
        
        if (j<5){
            flag=false;
        }
        for(int i=0; i<j; i++){
            char c1=s.charAt(i);
            if (c1=='.'){
                cnt++;
            }else if (cnt>=2){
                flag=false;
            }
            if(c1 != '.'&& c1<'0' || c1>'9'){
                flag=false;
            }
        }
         String lst=s.substring(j-1,j);
            if(lst.equals(".")){
                flag=false;
            }
            System.out.println(flag);
    }
}
