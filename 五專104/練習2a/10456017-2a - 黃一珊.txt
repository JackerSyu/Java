(1)
字串的長度是至少為2.
例如:
a5322121a  (正確)
a          (不正確)

class Main {
    public static void main(String[] args) {
        String s="a";
        int i=s.length();
        
        if(i>=2){
            System.out.println("通過");
        }else {
            System.out.println("不通過");
        }
    }
}

   (3-2)不容易閱讀: 具有特定意義的程式段落缺少分隔或註解, 或是同一行中擠進太多內容.                     
   (3-3)變數命名不適合: 未給予具有特別意義的變數符合其意義的名稱.

   (3-5)過於複雜的設計: 很明顯有更簡單的設計方式可以取代現有設計.
   (3-6)重覆或過於相似的程式段落: 程式中存在多個相同或相似段落, 明顯可整合但未整合.
   (3-7)沒有意義的指令: 存在不影響程式執行結果, 但又會被執行的程式敘述.
   *(3-8)程式撰寫其他建議.

123123  (不正確)

class Main {
    public static void main(String[] args) {
        String s="aaaaaaaaaaa";
        boolean flag=true;
        
        for(int i=0; i<s.length()-1;i++){
          char c1 =s.charAt(i);
          char c2 =s.charAt(i+1);
          
          if(c1!=c2 || s.length()<5 || s.length()>10){
             flag=false;
          }
        }
        System.out.println(flag);

(5)
字串長度至少為5, 內容只能有數字及小數點.
字串內容可以沒有小數點, 但如果內容有小數點, 只能有1個, 而且小數點之後一定要有其他數字.
例如:
54221     (正確)
543.1     (正確)
5321.     (不正確)
5.3.1     (不正確)

class Main {
  public static void main(String[] args) {
     String s="5.3.1";
    
     int cnt1=0;
     boolean flag=false;
    
        for(int i=0;i<s.length();i++){
          char c = s.charAt(i);
      
        if(c=='.'){
           cnt1++;
        }
        
        if(c>='0' && c<='9'|| c=='.'){
           flag=true;
        }else if(cnt1>1){ //小數點如果大於1則false
           flag=false;
        }
        
        String last = s.substring(s.length()-1,s.length());
        if(last.equals(".")){
           flag=false;
        }
        if(s.length()<5){
           flag=false;
        }
     }
        System.out.println(flag);
  }
}
