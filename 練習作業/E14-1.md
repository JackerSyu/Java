```java
方法說明:
(1) 平均成本 = (購入成本 + 行銷成本) / 進貨數量, 小數無條件進位.

(2) 定價 = 平均成本 * (1 + 利潤率), 小數無條件進位.
    利潤率的設定如下:
    (2-1) 若進貨數量 > 200, 利潤率 = 1.2
    (2-2) 若進貨數量 > 150 且 進貨數量 <= 200, 利潤率 = 1.25
    (2-3) 若進貨數量 > 100 且 進貨數量 <= 150, 利潤率 = 1.3
    (2-4) 若進貨數量 <= 100, 利潤率 = 1.4

(3) 輸出範例:    
    品名:粉彩筆, 進貨數量:140, 購入成本:7500元, 行銷成本:1500元, 平均成本:65元, 定價:85元
```

```java
import java.util.ArrayList;
import java.util.List;

import com.abc.Product;

public class Main {

        // 產生一個List
        List<Product> list = new ArrayList();

        }
```
