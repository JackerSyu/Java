```
   Java專案
       |__ <com.abc>
       |       |__ Pricing.java           (訂價類別, 抽象類別)
       |       |__ StorePricing.java      (商店訂價類別)       
       |       |__ WholesalePricing.java  (批發訂價類別)        
       |
       |__ Main.java 
```   


``` 
2. 類別繼承如下:

  
                             java.lang.Object 
                              (所有類別的父親)          
                                    /_\ 
                                     |  
                                     |  (繼承)  
                                     |
                                Pricing.java 
                              (訂價類別, 抽象類別)     
                                    /_\ 
                                     |
                                     |  (繼承)  
              _______________________|________________________  
             |                                                | 
             |                                                |
             |                                                |
      StorePricing.java                              WholesalePricing.java
      (商店訂價類別)                                      (批發訂價類別)                  
```

```
3. 批發訂價的[價格] = ((固定成本 + 變動成本) / 數量 ) / (1 - 毛利率) * 折扣, 小數無條件進位.  
   
4. 以下是[商店訂價]輸出範例:    
   商品名稱:粉彩筆, 固定成本:15,000元, 變動成本:75,000元, 數量:1400, 毛利率:0.2, 折扣:0.9, 訂價:72元
```
