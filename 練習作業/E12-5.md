```
1. 參考J12B, 建立的專案結構如下:

   Java專案
       |__ <com.abc>
       |       |__ Utility.java          
       |       |__ Employee.java            (員工類別)
       |       |__ PartTimeEmployee.java    (兼職員工類別)       
       |
       |__ Main.java 
       |
       |__ partTimeEmployee.csv  (輸入檔)
       |__ out.txt               (輸出檔)
```   


``` 
2. 類別繼承如下:
   
                                              java.lang.Object 
                                               (所有類別的父親)          
                                                     /_\ 
                                                      |  
                                                      |  (繼承)  
                                                      |
                                                Employee.java 
                                                  (員工類別)     
                                                     /_\ 
                                                      |
                                                      |  (繼承)  
              ________________________________________|_______________________________________  
             |                                        |                                       |
             |                                        |                                       |
             |                                        |                                       |   
   ExcellentEmployee.java                   OverseasEmployee.java                   PartTimeEmployee.java
       (優秀員工類別)                            (海外員工類別)                           (兼職員工類別) 
       (本練習未使用)                            (本練習未使用) 
```

```
3. 讀入partTimeEmployee.csv, 依序計算及輸出各兼職員工資料至out.txt

4. [兼職員工]的薪水計算方式為[兼職時數*每小時薪水]:
    如果是A職等, 每小時薪水 400元,
    如果是B職等, 每小時薪水 300元,
    如果是C職等, 每小時薪水 220元,
    其他, 每小時薪水 185元, 
  
5.  以下是[兼職員工]輸出範例:  
    編號:120001, 姓名:蔡家雅, 部門:人事部, 職等:A, 兼職時數:100, 每小時薪水:300, 薪水:30,000元
```
