# J14-1 執行緒

## (1) 執行緒們互相競爭電腦資源, 搶到就用, 一下子又被搶走.


```
     Thread.java    
     (執行緒類別)
         /_\
          | 
          |  (繼承)
          |
     MyThread.java   
 (自訂功能的執行緒類別)
```

<br/>

```
Java專案
   |__ <com.abc>
   |       |__ MyThread.java   
   |
   |__ Main.java
```


### (1) MyThread.java

```java
package com.abc;

//=====================================  
// 父親是一個執行緒類別. 
// 改寫父親的run()方法.
//=====================================  
public class MyThread extends Thread{ 
    //------------------------------
    // 建構元(1)
    //------------------------------ 
    public MyThread(String name){
        super(name);
    }

    //------------------------------
    // 建構元(2)
    //------------------------------ 
    public MyThread(){
        super();
    }
    
    //==============================  
    // 改寫父親的run()方法
    //==============================      
    @Override
    public void run(){
        for(int i = 0; i < 1000; i++){
            System.out.print(super.getName());
        }
    }
    //==============================       
}
```


### (2) Main.java

```java
import com.abc.MyThread;

public class Main {
    public static void main(String[] args) {
        //------------------------------
        // 生成2個執行緒物件
        //------------------------------
        MyThread t1 = new MyThread("小白");
        MyThread t2 = new MyThread("小黑");
 
        //------------------------------
        // 啟動執行緒物件
        //------------------------------
        t1.start();
        t2.start();
        //------------------------------
    }    
}
```
