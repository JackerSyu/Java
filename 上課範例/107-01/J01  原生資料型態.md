## J01 原生資料型態


### (1) 關於整數

```java
class Main {
    public static void main(String[] args) {
        int i=100;
        System.out.println(i);
    }
}
```

<br/>

### (2)關於浮點數

```java
class Main {
    public static void main(String[] args) {
        double i=12.345;
        System.out.println(i);
    }
}
```

<br/>

### (3)關於字元

```java
class Main {
    public static void main(String[] args) {
        char i='A';
        System.out.println(i);
    }
}
```

<br/>

### (4)關於布林

```java
class Main {
    public static void main(String[] args) {
        boolean i=true;
        boolean j=10<5;
        System.out.println(i);
        System.out.println(j);
    }
}
```

<br/>

### (5)型態轉換

```java
class Main {
    public static void main(String[] args) {
        double i=123.4567890123;
        float j=(float)i;
        System.out.println(j);
    }
}
```


<br/>

### (6)用Math類別中的方法

```java
class Main {
    public static void main(String[] args) {
        double i=Math.sqrt(15);
        System.out.println(i);
    }
}
```
