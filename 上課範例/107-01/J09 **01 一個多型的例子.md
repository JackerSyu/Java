# J09  **01 一個多型的例子

## Student類別及ForeignStudent類別, 都實作了Writeable介面

```
Java專案
   |__ <com.abc>
   |       |__ Writeable.java       (介面, 定義輸出資料的方法)
   |       |__ Student.java         (學生類別, 實作Writeable的類別)   
   |       |__ ForeignStudent.java  (外國學生類別, 實作Writeable的類別)   
   |
   |__ Main.java
```

<br/>

```
            Writeable.java 
                  | 
        __________|___________      
       |        (實作)        |
       |                      |
  Student.java        ForeignStudent.java  
```

### (1) Writeable.java

```java
package com.abc;

//----------------------------------------------------------
// 一個Writeable介面, 其中有一個未實作的output()方法,
// 由實作它的類別完成方法內容的設計, 其目的是為了多型.
//----------------------------------------------------------
public interface Writeable {
    public String output();    
}

```

<br/>

### (2) Student.java

```java
package com.abc;

public class Student implements Writeable{
    // 成員    
    private String stuNo;
    private String stuName;
    
    // 建構元(1)
    public Student(String stuNo, String stuName){
        this.stuNo = stuNo;
        this.stuName = stuName;
    }

    // 建構元(2)    
    public Student(){}

    // Getters    
    public String getStuNo(){return this.stuNo;}
    public String getStuname(){return this.stuName;}

    // Setters    
    public void setStuNo(String stuNo){this.stuNo = stuNo;}
    public void setStuName(String stuName){this.stuName = stuName;}
    
    //---------------------
    // 實作介面的方法    
    //---------------------
    @Override
    public String output(){
        return this.stuNo + "," + this.stuName;
    }
    //---------------------    
}
```

<br/>

### (3) ForeignStudent.java

```java
package com.abc;

public class ForeignStudent implements Writeable{
    // 成員    
    private String stuNo;
    private String stuName;
    private String country;
    
    // 建構元(1)
    public ForeignStudent(String stuNo, String stuName, String country){
        this.stuNo = stuNo;
        this.stuName = stuName;
        this.country = country;
    }

    // 建構元(2)    
    public ForeignStudent(){}

    // Getters    
    public String getStuNo(){return this.stuNo;}
    public String getStuname(){return this.stuName;}
    public String getCountry(){return this.country;}    

    // Setters    
    public void setStuNo(String stuNo){this.stuNo = stuNo;}
    public void setStuName(String stuName){this.stuName = stuName;}
    public void setCountry(String country){this.country = country;}    
    
    //---------------------
    // 實作介面的方法    
    //---------------------
    @Override
    public String output(){
        return this.stuNo + "," + this.stuName + "," + country;
    }
    //---------------------    
}
```

<br/>

### (4) Main.java

```java
import com.abc.Student;
import com.abc.ForeignStudent;
import com.abc.Writeable;

public class Main {
    public static void main(String[] args) {
        // 產生一個Student物件
        Student s1 = new Student("120001", "王小明");
        printInfo(s1);

        // 產生一個ForeignStudent物件        
        ForeignStudent s2 = new ForeignStudent("120002", "John", "美國");
        printInfo(s2);        
    }

    //------------------------------------------------------
    // 一個接收Writeable物件的方法
    //------------------------------------------------------
    public static void printInfo(Writeable writeable){
        System.out.println(writeable.output());
    }
    //------------------------------------------------------    
}
```
