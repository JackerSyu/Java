# J11__01 泛型-方法

## (1) 用 "T" 隱喻 "Type", 其型態在呼叫時才決定.

```
Java專案
   |
   |__ Main.java
```

### (1-1) Main.java

```java
import java.util.ArrayList;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        //---------------------------
        // 建立一個存放Integer的List
        //---------------------------        
        List<Integer> list1 = new ArrayList();
        
        list1.add(10);
        list1.add(20);
        list1.add(30);
        
        printData(list1);
 
        //---------------------------
        // 建立一個存放Boolean的List
        //---------------------------   
        List<Boolean> list2 = new ArrayList();
        
        list2.add(true);
        list2.add(false);
        list2.add(true);
        
        printData(list2);
        //---------------------------        
    }
 
    //----------------------------------------------
    // 這個方法可以接收儲存不同型態的List
    //----------------------------------------------  
    public static <T> void printData(List<T> list){
        list.forEach(s -> {
            System.out.println(s);
        });
    }    
}
```
