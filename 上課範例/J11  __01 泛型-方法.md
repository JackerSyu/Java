# J11__01 泛型-方法

## (1) 用 "T" 隱喻 "Type", 其型態在呼叫時才決定.

```
Java專案
   |
   |__ Main.java
```

### Main.java

```java
import java.util.ArrayList;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        //---------------------------
        // 建立一個存放Integer的List
        //---------------------------        
        List<Integer> list1 = new ArrayList();
        
        list1.add(10);
        list1.add(20);
        list1.add(30);
        
        printData(list1);
 
        //---------------------------
        // 建立一個存放Boolean的List
        //---------------------------   
        List<Boolean> list2 = new ArrayList();
        
        list2.add(true);
        list2.add(false);
        list2.add(true);
        
        printData(list2);
        //---------------------------        
    }
 
    //----------------------------------------------
    // 這個方法可以接收儲存不同型態的List
    //----------------------------------------------  
    public static <T> void printData(List<T> list){
        list.forEach(s -> {
            System.out.println(s);
        });
    }   
    //----------------------------------------------     
}
```



## (2) 用 "E" 強調處理的是一個未知的型態"Element".

```
Java專案
   |
   |__ Main.java
```

### Main.java

```java
import java.math.*;

public class Main {
    public static void main(String[] args) {
        //---------------------------
        // 以不同型別測試大小
        //---------------------------        
        printInfo(12, 35);
        printInfo(40, 32.5);
        printInfo("35.6", "51.3");
        printInfo("49", 36);
        printInfo(50, 50);
        //---------------------------        
    }
 
    //----------------------------------------------
    // 這個方法可以接收不同型態的參數
    //----------------------------------------------  
    public static <E> void printInfo(E data1, E data2){
        System.out.println(data1.getClass());
        System.out.println(data1);
            
        System.out.println(data2.getClass());
        System.out.println(data2);  
        
        System.out.println("---------------");   
    }   
    //----------------------------------------------     
}
```
